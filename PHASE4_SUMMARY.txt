TECHNICAL SPECIFICATIONS
================================================================================

NEURAL NETWORK ARCHITECTURE (per network):
  Input Layer:     7 features (S/K, T, σ, r, q, RSI, vol_ratio)
  LSTM Layer 1:    64 hidden units
  LSTM Layer 2:    64 hidden units
  Dense Layer 1:   32 units + ReLU
  Output Layer:    1 unit (option price)

ENSEMBLE CONFIGURATION:
  Number of Networks:     5
  Random Seeds:           Different per network
  Training Epochs:        50 (default)
  Batch Size:             32
  Learning Rate:          0.001
  Optimizer:              Adam

UNCERTAINTY QUANTIFICATION:
  Ensemble Disagreement:  σ_ensemble = std(predictions)
  Blending Weight:        w = sigmoid(-α·σ_ensemble + β)
  Final Price:            w·price_NN + (1-w)·price_BS
  Confidence Score:       Based on ensemble agreement

HESTON PARAMETERS:
  v0 (initial variance):         [0.01, 1.0]
  κ (mean reversion speed):      [0.1, 10]
  θ (long-term variance):        [0.01, 1.0]
  ξ (volatility of volatility):  [0.1, 2.0]
  ρ (correlation):               [-0.95, 0.0]
